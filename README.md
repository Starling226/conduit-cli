# Conduit

Conduit runs inproxy from [psiphon-tunnel-core](https://github.com/Psiphon-Labs/psiphon-tunnel-core). This repository targets a cross-platform CLI.

For more information about Conduit, [visit the web site](https://conduit.psiphon.ca).

## CLI App
To run cli in a development environment, you need a psiphon config file. This can be obtained from Psiphon at conduit-oss@psiphon.ca.

See the [cli](./cli) folder for more details about the Conduit CLI.
